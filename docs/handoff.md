# 海外客户开发项目说明

## 项目是什么

本项目用于寻找真实采购过目标产品的海外公司，并整理其有效联系方式，为后续客户开发和批量邮件触达提供可靠名单。

## 核心功能

- **寻找目标客户**：根据产品品类，从全球贸易数据中筛选具有真实采购记录的公司，优先覆盖欧美市场。
- **判断客户相关性**：结合实际贸易记录核对产品，排除名称相似但业务无关的公司。
- **获取有效联系方式**：整理公司邮箱、电话、官网等信息，并对邮箱和电话的有效性进行验证。
- **统一客户数据**：同一家公司只保留一条记录，同时保留其产品品类、贸易信息、联系方式和信息来源。
- **控制获客成本**：记录每次客户搜索和联系方式获取的费用，可持续评估投入产出。

## 当前成果

项目已覆盖胶带、遮蔽膜、刷子、猪毛刷、羊毛刷、PVC 护角条、塑料桶和油漆刷等品类，共整理 **270 家海外公司**：

- 270 家均拥有至少一种有效联系方式；
- 205 家拥有有效邮箱；
- 210 家拥有有效电话；
- 累计数据获取成本为 **¥387.10**，平均每获得一家可联系公司约 **¥1.43**。

## 项目价值

这套系统不是简单收集网络邮箱，而是先确认企业与目标产品存在真实贸易关系，再获取并验证联系方式。因此，最终名单同时具备客户相关性、联系方式有效性和信息可追溯性，可以直接用于销售筛选、客户分组和后续触达准备。
