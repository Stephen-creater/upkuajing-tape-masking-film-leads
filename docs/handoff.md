## 维护原则

- 不创建第二份客户总表；直接更新权威 Excel，历史由 Git 保留。
- Excel 业务展示统一使用中文：胶带、遮蔽膜、刷子、猪毛刷、羊毛刷、PVC护角条、塑料桶；搜索词与品类使用同一口径。
- 国家和已识别的产品术语优先显示中文；“经营范围（中文）”整列不得出现外文，空值统一显示“暂无经营范围信息”。经营范围原文保留在 `company-master.json` 的 `business_scope` 字段和 `data/raw/` 中。
- 公司名、联系方式、网址、报关品名和地址保留原文，原始 API 数据不改写。
- 新的 API 返回先落 `data/raw/`，再进入处理层和 Excel，不直接手改原始数据。
- 官网补充必须记录明确来源 URL，不用推测值。
- 跟进状态和负责人是 Excel 中的人工维护字段。
- 所有修改按小步提交，每一个可验证的小步都要 `commit + push`，不留本地未同步改动。
